# Global Tax Compliance & Exposure Analytics Platform

## Overview

Built a Tax Compliance & Exposure Analytics solution using Azure Databricks, PySpark, and Spark SQL.

The objective was to analyze tax transactions, filing compliance, and audit records to identify tax exposure and compliance risk across multiple countries.

---

## Tech Stack

- Azure Databricks
- PySpark
- Spark SQL
- Unity Catalog Volumes

---

## Dataset

- 100,000 Tax Transaction Records
- 10,000 Tax Filing Records
- 5,000 Audit Records

---

## Key KPIs

### Tax Exposure
Total tax liability by country.

### Compliance Percentage
Percentage of filings completed on time.

### Audit Risk
Average audit issues by country.

### Risk Classification
Countries categorized into:
- High Risk
- Medium Risk
- Low Risk

---

## Key Features

- Data ingestion using Databricks Volumes
- PySpark transformations
- Aggregations and joins
- Spark SQL analytics
- Tax risk dashboard creation

---

## Output

Final dashboard contains:

- Country
- Total Tax Exposure
- Compliance %
- Average Audit Issues
- Risk Level
